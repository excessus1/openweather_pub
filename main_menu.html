main_menu module

scripts.main_menu.api_logs_submenu()

Submenu to view recent API logs from various tables.

scripts.main_menu.clear_screen()

Clears the terminal screen.

Works cross-platform by checking the OS type and running the appropriate clear command.

scripts.main_menu.graph_menu(data, timestamps, data_alias)

Menu for generating graphs based on data and user selection.

Parameters:

  • data Dictionary containing the queried data (e.g., temp, humidity).

  • timestamps List of timestamps corresponding to the data.

  • data_alias Dictionary mapping user-friendly options to data keys in data.

scripts.main_menu.location_data_submenu()

Submenu for viewing location-specific data.

scripts.main_menu.main_menu()

Main menu for the CLI system.

scripts.main_menu.plot_line_chart(timestamps, values, title)

Plots a line chart with time on the x-axis.

scripts.main_menu.plot_wind_direction_radial(wind_directions)

Plots wind directions on a radial chart.

scripts.main_menu.plot_wind_rose(wind_directions, wind_speeds)

Plots a wind rose showing wind direction and speed.

Parameters:

  • wind_directions (list) List of wind directions in degrees (0360).

  • wind_speeds (list) List of corresponding wind speeds.

scripts.main_menu.plot_word_cloud(descriptions)

Generates a word cloud visualization for weather descriptions.

Queries the hourly_data and daily_summary_data tables for weather descriptions and creates a word cloud based on their frequency.

scripts.main_menu.recent_data_submenu()

Submenu for viewing recent data.

scripts.main_menu.tail_log(log_file)

Displays the last N lines of a specified log file, with user-specified scrollback.

Parameters:

log_file (str) Path to the log file to read.

Returns:

None

scripts.main_menu.view_api_call_types()

Displays all rows from api_call_types.

scripts.main_menu.view_api_script_tracking()

Displays all rows from api_script_tracking.

scripts.main_menu.view_location_daily_data()

Displays daily data for a specific location and date range.

scripts.main_menu.view_location_hourly_data()

Displays hourly data for a specific location and date/time range.

scripts.main_menu.view_locations()

Displays all locations stored in the database.

Queries the locations table and prints the details of each location in a user-friendly format.

scripts.main_menu.view_recent_api_calls()

Displays the most recent logs from api_calls.

scripts.main_menu.view_recent_daily_data()

Displays the most recent daily weather summaries.

Queries the daily_summary_data table to show the latest entries, including metrics such as high/low temperatures, precipitation totals, and weather descriptions.

scripts.main_menu.view_recent_hourly_data()

Displays the most recent hourly weather data in a human-friendly format.

Queries the hourly_data table and displays the latest 5 entries, including conversions for units like temperature, pressure, and wind speed.

scripts.main_menu.view_recent_logs()

Displays the most recent API logs.

Queries the api_calls table to show the latest API calls, including timestamps, response codes, and any relevant log messages.

scripts.main_menu.view_recent_sql_handling()

Displays the most recent logs from sql_handling.
